# File Organizer by Python

A simple Python script that automatically organizes files in a folder based on their extentions.It creates subfolders like Images,Documents,and others and then moves files into the right category.

# How it Works

-The script reads all files in the selected folder.
-It checks each file's extension like .jpg,.pdf,.txt,and etc.
-It creats the required subfolder automatically.
-Then it moves the file into the correct folder.

# Key Tech used

-Python 3.
-os,pathlib libraries,shutil.

# Features

-Automatically classifies files.
-Creates folders only when needed.
